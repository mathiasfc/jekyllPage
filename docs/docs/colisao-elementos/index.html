Colisão entre dois elementos

12/06/2017

Para apresentar como funciona a detecção de colisão entre dois objetos em javascript utilizando jquery, vou utilizar duas div’s quadradas inseridas em um plano 2D (x,y), lembrando que  a detecção de colisão pode ser aplicada para qualquer forma de objeto, desde que tenhamos as dimensões dos objetos em questão e suas respectivas posições. Esse algoritmo serve também para planos em 3D (x,y,z), obviamente que a complexidade da detecção aumentará, mas o raciocínio permanece o mesmo.

No exemplo em questão os dois objetos (div’s) vão possuir as seguintes dimensões, a primeira div (azul) possui 100 pixels de altura e 100 pixels de largura, a segunda div (vermelha) possui a metade do tamanho da primeira, logo 50 pixels de altura e 50 pixels de largura.

Posicionamento

O próprio jquery disponibiliza duas formas de obter a posição de um elemento:

  • Offset: Get the current coordinates of the first element in the set of matched elements, relative to the document.
  • Position: Get the current coordinates of the first element in the set of matched elements, relative to the offset parent.

O .offset() retorna a posição top e left do elemento em relação ao documento, diferentemente do .position() que retorna as posições em relação ao ancestral mais próximo. Não explicarei a fundo os métodos visto que não é a intenção deste post, para entender mais sobre eles basta acessar esses link’s da própria documentação: offset, position.

Colisão

A colisão nesse contexto que criamos, considerando somente estes dois elementos, vai acontecer quando as seguintes condições forem verdadeiras:

Na horizontal:

Quando a extremidade direita do elemento mais a esquerda (A2) for maior ou igual a extremidade esquerda do elemento mais a direita (V1) e menor ou igual a extremidade direita do elemento mais a direita (V2).

V1<=A2<=V2
